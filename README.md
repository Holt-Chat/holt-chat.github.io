Static HTML and CSS, no build step. Served live at [holt-chat.github.io](https://holt-chat.github.io) via GitHub Pages.

## Local preview

```sh
python serve.py
```

Opens on [http://localhost:3317](http://localhost:3317) (set the `PORT` env var to change it).
