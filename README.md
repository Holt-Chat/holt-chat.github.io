# Holt landing page

Marketing site for [Holt](https://github.com/Holt-Chat) — an end-to-end-encrypted chat app that's private but social.

Static HTML and CSS, no build step. Served live at https://holt-chat.github.io via GitHub Pages.

## Local preview

```sh
python serve.py
```

Opens on http://localhost:3317 (set the `PORT` env var to change it).
